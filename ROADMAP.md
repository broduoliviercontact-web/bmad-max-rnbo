# Roadmap BMAD-MAX-RNBO

## v1.0.0 - Foundation

Objectif : poser le cadre stable du framework.

- agents de base ;
- workflows de base ;
- templates de projet ;
- checklists ;
- knowledge rules ;
- scripts Python simples ;
- documentation d'installation et d'usage.

## v1.1.0 - Audit Maxpat avance

Objectif : ameliorer l'analyse automatique des fichiers maxpat.

- detection plus fine du fan-out ;
- detection des objets audio dangereux ;
- detection des objets inconnus ;
- detection de chemins absolus ;
- rapport Markdown complet ;
- score de lisibilite ;
- suggestions de refactor.

## v1.2.0 - RNBO Compatibility Layer

Objectif : enrichir les regles RNBO.

- base d'objets compatibles/incompatibles ;
- matrice de remplacement ;
- checklist export plus stricte ;
- template de moteur rnbo ;
- exemple de synth RNBO export web.

## v1.3.0 - Web Export Kit

Objectif : fournir un kit React/Vite minimal.

- loader RNBO JS ;
- composant React de controle de parametres ;
- gestion AudioContext ;
- affichage erreurs ;
- exemple exportable.

## v1.4.0 - M4L Device Workflow

Objectif : specialiser la methode pour Max for Live.

- workflow MIDI effect ;
- workflow audio effect ;
- workflow instrument ;
- regles Live parameters ;
- preset strategy ;
- limites Max for Live.

## v2.0.0 - Production System

Objectif : transformer BMAD-MAX-RNBO en veritable systeme de production.

- CLI plus complet ;
- integration Patcher API ;
- generation de patchs templates ;
- audit automatique plus strict ;
- exemples complets ;
- documentation pedagogique pour cours et ateliers.
