# RNBO Compatibility Checklist

- [ ] La boundary RNBO est claire.
- [ ] Le moteur RNBO fonctionne sans UI Max.
- [ ] Les paramètres sont exposés.
- [ ] Les valeurs par défaut sont définies.
- [ ] Les ranges sont définis.
- [ ] Les objets non compatibles sont hors RNBO.
- [ ] Les fichiers nécessaires sont embarqués ou documentés.
- [ ] Le niveau de sortie est sécurisé.
- [ ] L'export est testé.
